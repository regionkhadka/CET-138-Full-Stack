🎨 CSS

Cascading Style Sheets — controlling the appearance and layout of web pages

Before learning CSS properly, I used to wonder how websites looked so polished and professional. CSS was frustrating until I discovered Flexbox. Before that, I was trying to align everything with margins and it was a mess. This page shows the layout tools that finally made sense to me, especially the Grid system.

📦 1. The CSS Box Model

Every HTML element is a rectangular box. The box model defines how content, padding, border, and margin layer around it. Understanding this is essential for accurate layouts.

.box { margin: 20px; /* Space outside the border */ border: 4px solid green; padding: 20px; /* Space inside the border */ width: 200px; /* Content width */ }

↭ 2. Flexbox Layout

Flexbox is a one-dimensional layout method for arranging items in rows or columns. It makes it easy to distribute space and align items dynamically.

.container { display: flex; justify-content: space-between; /* Horizontal alignment */ align-items: center; /* Vertical alignment */ gap: 10px; }

▦ 3. CSS Grid Layout

CSS Grid is a two-dimensional layout system for the web, allowing control over both rows and columns simultaneously. It is ideal for page layouts.

.grid { display: grid; grid-template-columns: repeat(3, 1fr); gap: 10px; }

🏠 4. CSS Transitions & Animations

CSS can animate properties using transition for simple state changes and @keyframes for multi-step animations — without JavaScript.

@keyframes slideIn { from { transform: translateX(-60px); opacity: 0; } to { transform: translateX(0); opacity: 1; } } .box { animation: slideIn 1s ease; }

📱 5. Responsive Design with Media Queries

Media queries allow different styles to be applied at different screen sizes. This is the foundation of responsive, mobile-first design.

/* Mobile first — base styles for all screens */ .card { width: 100%; } @media (min-width: 768px) { .card { width: 50%; } } @media (min-width: 1024px) { .card { width: 33%; } }
This very portfolio is responsive! Resize your browser window to see the grid layout change between 1, 2, and 3 columns automatically.

🌟 6. CSS Custom Properties (Variables)

CSS variables allow you to store reusable values — like brand colours — in one place. Changing one variable updates every element that uses it.

:root { --primary-color: #006064; --font-size-lg: 1.25rem; } .button { background: var(--primary-color); font-size: var(--font-size-lg); }
