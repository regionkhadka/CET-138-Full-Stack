📄 HTML Structure

The backbone of web content

HTML was the first thing I learned on this course and it forms the backbone of everything we build. Without HTML there is no structure — no headings, no forms, no tables. Every website we visit is built on HTML, even the most complex ones. In this section I've put together examples of the HTML concepts I've found most useful and interesting.

1. Semantic Elements

Semantic elements clearly describe their meaning to both the browser and the developer. Using them improves accessibility and search engine optimisation.

Why it matters: Screen readers and search engines understand the page structure better. A <nav> tells assistive technology "this is navigation" — a generic <div> does not.
