🌐 What is Full Stack Development?

Understanding every layer of a modern web application

Definition

When I first started this course, I thought web development was just HTML and CSS. Full stack development is actually much more, it means being able to work on every part of a web application, from the button a user clicks, all the way down to the database storing their data. A full stack developer connects all these pieces together.

The term "stack" refers to the collection of technologies used together to build a complete application. Common stacks include:

  • MEAN — MongoDB, Express, Angular, Node.js
  • MERN — MongoDB, Express, React, Node.js
  • LAMP — Linux, Apache, MySQL, PHP
  • JAMstack — JavaScript, APIs, Markup
The Three Layers
🌐 Frontend (Client Side)
What the user sees and interacts with in their browser.
HTML CSS JavaScript React
↕ HTTP Requests & Responses
⚙️ Backend (Server Side)
Business logic, authentication, data processing.
Node.js PHP Python Express
↕ SQL / NoSQL Queries
🗄 Database
Persistent data storage and retrieval.
MySQL MongoDB PostgreSQL

How a Request Flows Through the Stack

1. User makes a request

A user visits a URL or clicks a button in their browser. The browser sends an HTTP request to the server. Example: GET /products

2. Server processes the request

The backend receives the request, applies business logic, queries the database if needed, and prepares a response. Example: fetch all products from the database.

3. Response rendered to user

The server sends back HTML, JSON, or other data. The browser renders it. JavaScript may update the page dynamically without a full reload (AJAX/fetch).

📄 A Simple Full Stack Example

This shows how a frontend fetch request talks to a backend API, which queries a database and returns JSON data.

// FRONTEND — JavaScript fetch call fetch('/api/users') .then(res => res.json()) .then(data => console.log(data)); // BACKEND — Node.js/Express route app.get('/api/users', (req, res) => { db.query('SELECT * FROM users', (result) => { res.json(result); }); });

Backend Developer Server-side Node.js, PHP, Python, Java APIs, auth, business logic, databases
Full Stack Developer Both layers All of the above End-to-end development of features
