⚙️ Bootstrap Framework

The world's most popular CSS framework — responsive, component-rich, and utility-first

Bootstrap saved me a huge amount of time when I started building more complex pages. Instead of writing all my layout CSS from scratch, Bootstrap gives me a grid system and ready-made components that just work. This entire portfolio actually uses Bootstrap 5 — the navbar, cards, buttons and grid are all Bootstrap components.

▦ 1. The Bootstrap Grid System

Bootstrap's grid is based on a 12-column flexible layout. Columns are defined with classes like col-md-6 (half width on medium screens) and automatically stack on mobile.

<div class="row"> <div class="col-md-4">One third</div> <div class="col-md-4">One third</div> <div class="col-md-4">One third</div> </div>

🗂 3. Bootstrap Modal

Modals are overlay dialogs triggered by buttons. Bootstrap handles the show/hide behaviour via JavaScript — no custom JS needed.

<button class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#myModal">Open</button>

🧰 4. Utility Classes

Bootstrap's utility classes allow rapid styling without writing custom CSS. They cover spacing, typography, colours, display, and more.

mt-3 — margin-top: 1rem p-4 — padding: 1.5rem text-center — text-align: center fw-bold — font-weight: bold d-flex — display: flex gap-3 — gap: 1rem rounded-pill — border-radius: 50rem shadow-sm — small box shadow
rounded-pill shadow-sm fw-bold text-danger fs-5
