⚡ JavaScript

The programming language of the web — making pages dynamic and interactive

JavaScript (JS) is the only programming language that runs natively in web browsers. It allows developers to manipulate the DOM, respond to user events, fetch data from APIs, validate forms, and build complex single-page applications. It also runs on the server with Node.js.

1. DOM Manipulation

The DOM (Document Object Model) is a tree representation of the HTML page. JavaScript can select, read, modify, add, or remove any element in real time.

    4. Fetch API

    The fetch() function is used to make HTTP requests to APIs. It returns a Promise, which is handled with .then() or async/await. This is fundamental to dynamic, data-driven web apps.

    5. Local Storage

    Local Storage allows data to persist in the browser between sessions. It stores key-value pairs as strings and is useful for saving user preferences or draft content.
